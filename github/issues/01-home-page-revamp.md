# Home Page Revamp

## 📖 Overview

Redesign the Chapters home page to provide users with a welcoming, intuitive starting point for managing their journals. The homepage should make it easy to understand the current state of journals, quickly create new ones, and continue writing.

---

## 🎯 Goal

Improve the first-time and returning user experience by making the homepage more informative, visually appealing, and action-oriented.

---

## User Story

**As a** journal user

**I want** to quickly see my journals and create new ones

**So that** I can start writing with minimal friction.

---

## Scope

### Redesign Layout

- Redesign the homepage layout
- Improve spacing and typography
- Modernize overall visual appearance

### Empty State

When no journals exist:

- Show friendly illustration or icon
- Explain what Chapters is
- Display a prominent **Create Journal** CTA

### Journal Overview

Display:

- Total Journals
- Total Entries
- Recently Updated Journal
- Last Entry Date

### Navigation Improvements

- Better journal list
- Highlight selected journal
- Improve scrolling behavior
- Better responsive layout

### Primary Actions

Provide quick actions:

- Create New Journal
- Open Recent Journal
